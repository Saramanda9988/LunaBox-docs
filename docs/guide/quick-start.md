# 快速开始

安装并启动 LunaBox 后，您可以按照以下步骤快速上手。

## 1. 添加游戏

- 点击主界面上的 **添加按钮**。
- 您可以手动选择游戏的可执行文件（.exe）。
- 或者选择 **批量导入**，选择包含多个游戏的文件夹进行扫描。
- 还支持从 **PotatoVN** 或 **Playnite** 等其他平台导入数据。

## 2. 管理分类

- 为了更好地组织您的游戏库，您可以创建 **自定义分类**。
- 右键点击游戏或在设置中管理分类标签。
- 将游戏拖拽或分配到相应的分类中。

## 3. 开始游玩

- 在游戏库中找到您想玩的游戏。
- 点击 **启动按钮**。
- LunaBox 会在后台自动追踪您的 **游玩时长**。
- 游戏结束后，时长数据会自动更新。

## 4. 查看统计与分享

- 切换到 **统计页面**。
- 您可以看到按日、周、月、年统计的详细游玩数据和图表。
- 点击 **导出** 按钮，可以生成精美的统计卡片，与朋友分享您的游戏历程。
- 使用 **AI 分析** 功能，获取对您游玩习惯的独特见解。
