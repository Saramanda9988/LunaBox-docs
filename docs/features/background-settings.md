# 背景图设置

LunaBox 允许您自定义应用界面的背景图片，并提供丰富的个性化调节选项。

## 1. 启用自定义背景

1. 进入 **设置** 页面。
2. 找到 **外观设置**。

![](/image/background-settings-1.png)

::: tip 
智能主题适配
开启自定义背景后，LunaBox 会自动检测图片的亮度，并智能切换应用的 **亮色/暗色主题**，以确保文字和图标清晰可见。
:::
::: warning 注意事项
如果想更换背景图片，请先点击**清除**按钮，再点击**选择**按钮上传本地图片。
:::

## 2. 选择与裁剪图片

- 点击 **选择** 按钮上传本地图片。
- 支持格式：PNG, JPG, JPEG, GIF, WebP, BMP。
- **裁剪功能**：上传后会自动通过裁剪工具，允许您选择想要作为背景的区域。

## 3. 视觉效果调节

您可以根据喜好微调背景图的显示效果：

- **背景模糊度 (Blur)**
  - 调节背景图片的模糊程度。
  - 增加模糊度可以减少背景对文字内容的干扰，使界面更易读。

- **遮罩不透明度 (Opacity)**
  - 调节覆盖在背景图上的遮罩层浓度。
  - **数值越低 (向左)**：背景图越清晰鲜艳。
  - **数值越高 (向右)**：背景图越淡，背景色越实（更接近纯色背景）。

## 4. 隐藏首页游戏封面

- 开启 **隐藏首页游戏封面** 选项。
- 启用后，首页将不再显示"上一次游玩"的大幅游戏封面图，而是完全展示您的自定义背景，提供更沉浸的视觉体验。

## 预览

设置界面提供实时预览区域，您可以在此即时查看到模糊度和透明度调整后的效果。

## 启用自定义背景
点击**启用自定义背景**开关，即可启用自定义背景。
