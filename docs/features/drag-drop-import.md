# 拖拽导入 (Drag & Drop Import)

LunaBox 支持最直观的导入方式——拖拽导入。您可以直接将一个或多个游戏文件夹拖入应用窗口，系统将自动识别并引导您完成导入流程。

## 1. 触发方式

在 LunaBox 的任意界面，直接将 **游戏文件夹** 拖入窗口区域。

- 支持同时拖入多个文件夹。
![alt text](/image/drag-import-1.png)
- 系统会自动扫描文件夹内的可执行文件 (`.exe`) 和目录结构。

## 2. 导入流程

拖入文件后，将弹出 **拖拽导入** 界面，流程如下：

### 第一步：文件识别与预览
![alt text](/image/drag-import-2.png)
系统会列出识别到的所有潜在游戏文件夹。在此阶段，您可以：

- **确认启动文件**：如果一个文件夹内有多个 `.exe` 文件（如 `Game.exe`, `Uninstall.exe`），您可以在下拉菜单中选择正确的游戏主程序。
- **编辑搜索名称**：系统默认使用文件夹名称作为搜索关键词。如果文件夹名称包含版本号或其他无关字符（如 `[v1.0] GameName`），您可以手动修改为纯净的游戏名称，以便提高搜索准确率。
- **筛选导入项**：取消勾选不想导入的项目。

### 第二步：匹配元数据
点击 **匹配元数据** 按钮。

- 系统将根据“搜索名称”自动从 Bangumi、VNDB、月幕 Galgame 等数据源抓取游戏封面、简介、开发商等信息。
- **匹配状态**：
  - <span style="color:green">✔ 已匹配</span>：成功找到元数据。
  - <span style="color:orange">⚠ 未找到</span>：未搜索到相关游戏。
  - <span style="color:red">✖ 错误</span>：网络或其他错误。

### 第三步：人工修正 (可选)
对于匹配结果不满意或未匹配的项目：
1. 点击右侧的 **✏ (编辑)** 按钮。
2. **手动搜索**：输入更准确的关键词重新搜索。
3. **指定数据源 ID**：如果您知道游戏在 Bangumi 或 VNDB 的 ID，可以直接输入 ID 进行精确绑定。

### 第四步：确认导入
确认所有信息无误后，点击 **导入 [N] 个游戏** 按钮。系统将把游戏添加到您的库中，并下载封面图片。

::: info
- 在点击“匹配元数据”前，简单检查一下识别出的“搜索名称”，删除多余的字符（如 `v1.2`, `汉化版` 等），能显著提高匹配成功率。
- 拖拽导入非常适合批量添加本地已有的游戏库。
:::
