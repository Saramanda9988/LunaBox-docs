 # 功能详解

LunaBox 提供了丰富的功能来增强您的视觉小说管理体验。

## 游戏管理

LunaBox 的核心是高效的游戏库管理。

- **多种导入方式**：支持[手动添加](./manual-add.md)、批量导入文件夹以及从第三方平台迁移。
- **自定义元数据**：您可以编辑游戏的封面、标题、开发者等信息。
- **状态标记**：标记游戏的通关状态（如：未开始、进行中、已完成）。
- **丰富的搜索与筛选**：通过关键词快速查找游戏，或通过分类进行筛选。

## 游玩统计

LunaBox 提供了详尽的数据统计功能。

- **多维度视图**：支持日、周、月、年视图，全方位展示游玩趋势。
- **可视化图表**：使用直观的柱状图和折线图展示数据。
- **数据导出**：一键生成分享卡片。

## AI 分析

利用 AI 技术，LunaBox 能为您生成有趣的个性化报告。

- **风格分析**：AI 分析您偏好的游戏类型和题材。
- **习惯总结**：总结您的游玩时间段和习惯。
- **趣味点评**：生成幽默风趣的评语。
