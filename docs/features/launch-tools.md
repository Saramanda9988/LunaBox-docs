# 启动工具配置 (Launch Tools)

LunaBox 支持集成第三方启动工具，以优化游戏体验。目前支持 **Locale Emulator**（解决乱码/转区问题）和 **Magpie**（全屏缩放/FSR）。

## 1. 配置启动工具路径

在使用这些功能之前，您需要在全局设置中指定工具的可执行文件路径。

1. 进入 **设置 (Settings)** 页面。
2. 找到 **游玩配置** -> **游戏启动工具** 区域。
3. 分别配置以下路径：
   - **Locale Emulator 路径**: 选择 `LEProc.exe` 文件。
   - **Magpie 路径**: 选择 `Magpie.exe` 文件。

::: tip
配置完成后，您就可以在单个游戏的启动配置页面中启用这些功能。
![alt text](/image/launch-tools-1.png)
:::

## 2. Locale Emulator (转区启动)

Locale Emulator 可以模拟特定语言环境（如日文环境）来运行游戏，通常用于解决日文游戏在非日文系统上出现的乱码或无法运行问题。

### 启用方法
1. 在游戏详情页，选择 **启动配置**。
2. 在启动配置界面的 **启动工具** 区域，找到 **使用 Locale Emulator 转区启动**。
3. 开启开关。

### 运行机制
- 游戏将通过 `LEProc.exe` 启动。
- 此时 LunaBox 会调用配置的 LE 路径，并将游戏可执行文件作为参数传入。

## 3. Magpie (超分辨率/缩放)

Magpie 是一个窗口全屏化工具，支持多种缩放算法（如 FSR, Anime4K），可以将不支持全屏或分辨率较低的游戏高质量地放大到全屏。

### 启用方法
1. 在游戏详情页，选择 **启动配置**。
2. 在启动配置界面的 **启动工具** 区域，找到 **使用 Magpie 超分辨率缩放**。
3. 开启开关。

### 运行机制
- 启动游戏时，LunaBox 会自动检测 Magpie 是否正在运行。
- 如果 Magpie 未运行，LunaBox 会尝试以托盘模式 (`-t`) 启动 Magpie。
- 游戏启动后，您可以使用 Magpie 的快捷键将游戏窗口全屏化。

::: warning 注意事项
- 确保您选择的路径是正确的 `LEProc.exe` 和 `Magpie.exe`，选择错误的文件将导致游戏无法启动。
- 如果 Magpie 启动失败，请检查 Magpie 是否需要管理员权限或有其他兼容性问题。
:::
