# 自动检测游玩进程

有些游戏会先启动一个「启动器 / 外壳进程」，随后真正的游戏进程才出现；如果只监控启动器，LunaBox 可能会误判“游戏已退出”，导致计时不准确。

为了解决这个问题，LunaBox 提供了 **自动进程检测**：当检测到启动器进程很快结束时，会提示你从当前运行的进程列表中选择「真正的游戏进程」，并将其用于后续监控与计时。

## 1. 在哪里开启

1. 打开 **设置 (Settings)**。
2. 找到 **游玩配置**。
3. 开启 **自动进程检测**。

![alt text](/image/auto-detect-1.png)

> 默认建议开启：适合带启动器、平台启动、或会拉起子进程的游戏。

## 2. 触发时机与流程

当你启动游戏后：

- 如果启动的可执行程序（启动器）很快退出
- 且系统检测到你可能启动的不是实际游戏进程

LunaBox 会弹出 **选择游戏进程** 窗口：

1. 自动加载当前正在运行的进程列表
2. 你可以使用搜索框快速筛选
3. 选择实际运行的游戏进程（通常是 `xxx.exe`）
4. 点击 **确认选择**，LunaBox 会开始按该进程进行监控与计时

![alt text](/image/auto-detect-2.png)

## 3. 如何避免每次都弹窗

如果你已经知道“真正的游戏进程名”，可以在单个游戏里预先填写：

1. 进入游戏详情页
2. 打开 **启动配置**
3. 在 **实际游戏进程** 中填写或点击 **选择** 选择 exe

![alt text](/image/auto-detect-3.png)

这样在启动时会优先使用已保存的进程名进行监控，减少每次手动选择。

## 4. 注意事项

- 进程名需要包含 `.exe` 后缀（例如 `Game.exe`）。
- 如果你不确定选哪个：优先选择看起来像游戏本体、且在游戏运行期间一直存在的进程。
- 如果取消选择：本次启动将不会使用你手动指定的进程名（可能回退为监控启动器/已保存配置）。
