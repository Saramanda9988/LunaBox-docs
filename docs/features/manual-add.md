# 手动添加游戏

LunaBox 支持手动添加游戏，通过选择游戏的可执行文件并自动搜索元数据来创建游戏条目。请按照以下步骤操作：

## 1. 开始添加游戏
- 在游戏库页面，点击 **添加游戏** 按钮，选择 **手动添加** 选项。
- 或者直接点击主界面上的 "手动添加" 按钮。

## 2. 选择游戏可执行文件
- 在弹出的文件选择对话框中，浏览到游戏安装目录。
- 选择游戏的主可执行文件（.exe 文件）。
- 系统会自动检测并填充游戏路径。

## 3. 搜索和获取元数据
- 系统会使用游戏文件名自动搜索元数据。
- 搜索来源按优先级排序：Bangumi > VNDB > 月幕gal。
- 您可以在搜索框中修改搜索关键词以获得更准确的结果。

## 4. 选择匹配的游戏
- 搜索结果会显示在列表中，包含游戏名称、封面图片、开发商等信息。
- 从结果列表中选择与您添加的游戏匹配的条目。
- 如果没有找到匹配的游戏，您可以选择跳过元数据获取，仅添加基本游戏信息。

## 5. 配置游戏信息
- 确认或修改游戏名称、开发商、描述等信息。
- 设置游戏的保存路径（如果适用）。
- 上传或更改游戏封面图片。

## 6. 完成添加
- 点击 **添加游戏** 按钮完成添加过程。
- 游戏将出现在您的游戏库中，可以随时启动和管理。

## 注意事项
- 推荐选择游戏的主程序文件，而非安装程序或卸载程序。
- 元数据搜索需要网络连接。
- 如果游戏名称包含特殊字符或非中文字符，可能需要手动修改搜索关键词以获得更好的结果。
- 游戏的启动路径将保存在数据库中，确保游戏文件在原位置保持不变。
